Receipt Transformer

A Word From The Developer -

This is a handy tool for quickly turning transactional metadata into a pile of DPCIs that are ready to be pasted into TCM.

Some important things to note:

  • Always double check before submission.
  • There may be too many or too few entries.
  • I am not responsible for incorrect evidence in TCM due to a bug in this software.

That said, it's pretty stable. At the time of this writing there is a known bug on transactions involving Cartwheel which causes some items to show up twice, as well as the creation of some incorrect DPCIs. There may also be a bug related to paper bag fees but I need to do more testing to be sure.
These bugs aside, I've still found this to be useful in saving time. In the future I'll provide a method for bug reporting as well as instructions and other documentation.
